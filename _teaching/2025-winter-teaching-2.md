This course is part of the BSc degree in Computational Linguistics. It covers basics of programming with focus on:

- Object-oriented 
	- Understanding and correctly using classes
    - Inheritance
    - Encapsulation
    - Abstraction
    - Polymorphism
	- Documentation
	- Code Review
- Data structures
